# 如何上手

1. **访问官网**：https://midjourney.com（国内需科学上网）
2. **注册账号**：用邮箱注册，选择订阅方案（推荐Standard $30/月）
3. **开始生成**：在Web应用中输入描述词（prompt），等待出图
4. **优化工作流**：探索阶段用Draft Mode（便宜快），确定方向后切标准模式出精稿
5. **学习Prompt**：Midjourney对prompt理解力强，但学习写好prompt能大幅提升出图质量

**省钱技巧**：Standard方案（$30/月）包含无限Relax Mode，等待2-5分钟即可无限生成，大多数用户的最佳选择。

# 编辑点评

Midjourney是我用过的最"有品味"的AI绘画工具。它的图不是最"准"的（手指偶尔还是有问题），但绝对是最"好看"的。

最大的坑是**没有免费版**——最低$10/月起，而且对国内用户来说支付和访问都有门槛。但如果你是靠视觉吃饭的（设计师、营销、自媒体），这$30/月是最值得的AI投资之一。

另外一个坑是**文字渲染几乎不可用**——如果你需要在图片里加文字，老老实实用DALL-E 3或者后期PS加上去，别浪费GPU时间。

竞品对比：DALL-E 3文字渲染好但审美一般，Stable Diffusion免费但上手难，Adobe Firefly商业安全但创意感不足。Midjourney就是"审美天花板"这个定位，短期内无人能撼动。
