# 编辑点评

Claude是我个人最常用的AI助手。原因很简单：它不说废话。

ChatGPT越来越像一个"点头型朋友"——你说什么它都附和。Claude不一样，它会直接告诉你"你这个想法有问题"。这种"批判性思维"在需要深度思考的场景里非常宝贵。

最大的坑是**配额不透明**——Pro用户经常遇到"突然不能用"的情况，Anthropic的解释总是含糊其辞。建议重度用户直接上Max，别省这个钱。

国产替代：Kimi（长文档中文更强）、通义千问（免费）。但论"敢说真话"和代码能力，Claude目前还是独一档。
