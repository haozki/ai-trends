## 使用说明

1. **LOGO**: 每个项目的 `asset/logo.*` 可直接用于文章配图
2. **截图**: `screenshot-1.png` 是官网首屏截图，可用作文章配图
3. **补充截图**: 建议手动补充产品功能界面截图（如对话界面、生成结果等）
4. **公众号上传**: 所有图片需手动上传到微信图床，不支持外链
5. **新增项目**: 复制 `_template/info.md` 模板开始

## 命名规范

- `logo.*` — 项目品牌标识
- `screenshot-N.png` — 产品界面截图
- `promo-N.png` — 官方宣传图
- 其他资源按用途命名
