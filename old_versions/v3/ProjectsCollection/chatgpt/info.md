# 如何上手

1. **访问官网**：https://chat.openai.com（国内需科学上网）
2. **注册账号**：用邮箱或Google账号注册，免费版可直接使用
3. **选择模型**：免费用户用GPT-4o mini，付费用户可选GPT-5
4. **开始对话**：直接输入问题，支持文字、图片、文件上传
5. **探索GPTs**：点击"Explore GPTs"发现专用工具，比如"Canva"做图、"Wolfram"算数学

**省钱提示**：每天使用超过20次再考虑Plus，偶尔用免费版足够。

# 编辑点评

ChatGPT不是完美的，但它是目前最"全能"的AI助手。如果你只打算用一个AI工具，ChatGPT是最安全的选择——它不一定在某项上最强，但几乎没有明显短板。

最大的坑是"谄媚倾向"——它太想让你满意，有时会附和你错误的观点。解决办法：提示词里加一句"请批判性思考，指出我观点里的问题"。

国产替代推荐：Kimi（长文档）、通义千问（免费好用）、豆包（日常对话）。但论生态和插件丰富度，ChatGPT目前还是独一档。
