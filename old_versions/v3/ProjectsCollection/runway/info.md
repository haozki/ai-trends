# 核心功能亮点

1. **Multi Motion Brush多重运动笔刷** - 在静态图片上圈出不同区域，每个区域独立指定运动方向和速度，远超简单Ken Burns效果
2. **Act-One角色动画捕捉** - 用演员表演视频驱动AI虚拟角色，无需专业动捕设备，一台摄像头搞定
3. **精准镜头控制** - 支持推镜、拉镜、平移、旋转等专业摄影术语指令，镜头控制能力同类最强
4. **Video to Video风格转换** - 上传实拍视频转换为油画/动漫/赛博朋克等风格，MV和广告创意利器
5. **复杂视觉特效** - 爆炸、火焰、烟雾、液体飞溅、霓虹灯光等特效处理质量领先

# 如何上手

1. **访问官网**：https://runwaycn.com（国内需科学上网）
2. **注册账号**：用邮箱注册，免费版有125积分（约25秒720p视频）
3. **选择方案**：独立创作者推荐Standard $15/月，专业用户推荐Pro $35/月
4. **开始生成**：输入文字描述或上传参考图片，选择模型（Gen-3 Alpha或Turbo），生成视频
5. **进阶技巧**：探索阶段用Turbo模型（更便宜更快），确定方向后用标准模型出精稿

**注意事项**：积分每月清零不累积，团队共享积分池需协调使用，避免快速耗尽。

# 编辑点评

Runway是我见过的"最不急着讨好用户"的AI视频工具。别人在卷时长、卷免费，它在卷"专业可控性"。这种定位反而让它成了好莱坞的选择。

最大的坑是**视频时长只有10秒**——如果你需要生成3分钟的长视频，Runway不适合你，直接看Kling。另一个坑是**国内访问难**，需要稳定的VPN。

选Runway还是Kling/Sora？简单判断：要专业镜头控制和特效→Runway；要长视频→Kling；要物理真实感→Sora。
