# 编辑点评

通义千问是"免费+全面"的典型代表。它不是任何单项的冠军，但在"免费"这个前提下，它什么都有一点——文本、图片、文档、钉钉集成——这对不想花钱的用户来说非常友好。

最大的优势是**阿里生态**：如果你公司用钉钉、团队用阿里云，通义的整合体验远超其他AI。最大的劣势是**创意和推理偏弱**——写文章不如Claude和Kimi，写代码不如DeepSeek和Claude，推理不如Kimi k1.5。

选通义千问还是Kimi？简单判断：需要免费+全面→通义千问；需要长文档+深度推理→Kimi。两个都免费，都试试不吃亏。
