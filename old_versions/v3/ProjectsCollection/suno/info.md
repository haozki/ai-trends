# 编辑点评

Suno是"AI+音乐"赛道里最有意思的产品——不是因为它的音乐质量能和专业制作人比，而是因为它让"音乐创作"这件事从专业技能变成了人人可玩的游戏。

它的核心价值不是"替代音乐人"，而是"让你能创作以前创不了的东西"。短视频配乐、播客BGM、朋友聚会的趣味创作——这些场景Suno比任何工具都好用。

最大的坑是**积分不结转**——每月清零，用不完就浪费了。另外歌词质量不稳定，偶尔会有重复或脱节的问题。

竞品对比：Udio（人声质量略逊，但免费额度更多）、网易天音（国产，中文效果更好）。论人声逼真度，Suno目前还是领先。
